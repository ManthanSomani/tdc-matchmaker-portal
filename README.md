Project Overview:-

The Date Crew is a full-stack matchmaking platform designed to bridge the gap between traditional rule-based filtering and modern AI-driven insights. By combining structured data analysis with semantic understanding, the platform provides users with highly compatible matches and personalized communication drafts.

Tech Choices:-

The architecture is built for scalability and performance. The frontend is a responsive React application styled with Tailwind CSS. The backend utilizes Node.js and Express, chosen for its non-blocking I/O, which is essential for handling multiple API requests. Because heavy AI models often cause memory exhaustion on free-tier hosting, the NLP similarity engine was decoupled into a Python/FastAPI microservice hosted on Hugging Face Spaces. This distributed approach ensures the primary backend remains lightweight while offloading computationally expensive vector operations.

Matching Logic & AI Integration:-

The matching logic employs a two-tier scoring system. First, the backend applies hard filtering based on core requirements (e.g., gender, age, religion). Second, it applies a "Soft-Alignment" score generated by a Sentence-Transformers model (all-MiniLM-L6-v2). This model converts job designations into mathematical vectors, calculating the cosine similarity between the user and potential matches. To add a layer of personalization, we integrate the Groq API (Llama 3.1) to process match data and synthesize warm, conversational introductory emails, effectively automating the matchmaker's "cold outreach" process.

Assumptions Made:-

This project assumes that professional designation is a primary proxy for lifestyle compatibility. We also assume that users are willing to provide structured profile data, which is essential for the initial filtering layer. Finally, the system operates on the assumption that the Python NLP microservice maintains high availability, allowing the Node.js backend to reliably fetch similarity scores for every match request initiated by the user.

Tech Stack:-

Frontend: React, Tailwind, Lucide

Backend: Node.js, Express, dotenv

AI Engine: Python, FastAPI, Sentence-Transformers

LLM API: Groq (Llama 3.1 8B)

Deployment: Render (Backend), Vercel (Frontend), Hugging Face (AI Engine)
